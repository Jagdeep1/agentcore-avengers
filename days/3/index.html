Daily Challenge

Day 3: Infinity War

In Infinity War, the heroes needed to gather the Infinity Stones - powerful artifacts that together grant ultimate capability. Today, we gather the capabilities that make your agent truly powerful: Memory and Tools.

Remember Day 1’s disaster? Your agent forgot everything between requests. Today, we fix that - and add powerful new abilities.

What You’ll Build

Today your agent gains:

  • Persistent Memory: Remember conversations and learn from interactions
  • Code Interpreter: Execute Python code for analysis and computation
  • Browser Tool: Navigate websites and automate web tasks

The Power Combination

Memory + Tools transforms your agent from a simple chatbot into a capable assistant that can remember context, execute code, browse the web, and accomplish complex multi-step tasks.

Key Concepts

AgentCore Memory

Short-Term Memory: Captures turn-by-turn interactions within a session

  • Maintains immediate conversation context
  • Users don’t need to repeat information
  • Automatically managed per session

Long-Term Memory: Extracts and stores insights across sessions

  • User preferences and facts
  • Conversation summaries
  • Semantic retrieval of relevant context

Built-in Tools

Code Interpreter: Secure Python execution environment

  • Data analysis and visualization
  • Complex calculations
  • File processing

Browser Tool: Web automation capabilities

  • Navigate to URLs
  • Interact with web elements
  • Capture screenshots

Prerequisites Check

Before proceeding, ensure you have:

  • Completed Day 2 (Identity + Gateway configured)
  • Agent deployed to AgentCore Runtime
  • Understanding of session management
  • Sample data for testing (CSV, JSON files)

Tool Safety

Code Interpreter and Browser are powerful tools. Today we’ll use them freely to understand their capabilities. Day 4 adds the guardrails for safe production use.

Today’s Controlled Disaster

CONTROLLED DISASTER

The Runaway Agent

Symptom

Your agent executes rm -rf commands, infinite loops, or navigates to malicious websites without any restrictions.

Root Cause

Unrestricted tool access - the agent can execute any code or visit any site.

The Fix

Day 4 introduces Guardrails to constrain agent behavior and prevent dangerous actions.

Learn more in Day 4 →

This disaster demonstrates why powerful tools require guardrails. By the end of Day 4, your agent will have both power AND safety.
